# 北京二中经开区学校论坛 — 前端重构设计规划（PRD）

## 一、设计愿景与理念

### 1.1 核心定位
打造一所**百年名校气质的数字化校园社区**。不是普通的"论坛模板"，而是承载学校精神、传递文化底蕴的线上空间。设计应当像一本精装校刊——有分量、有细节、有温度。

### 1.2 设计关键词
- **庄重而不沉闷** — 教育机构的严谨 + 青春校园的活力
- **质感优先** — 真实的材质感、纸张纹理、印刷级排版
- **文化自信** — 校徽、校训不是装饰，而是视觉核心
- **信息丰盈** — 每个页面都有层次，不空洞

### 1.3 设计方向：学院精装杂志风（Academic Editorial）
参考《The New Yorker》的排版气质 + 顶尖大学官网（Harvard、Tsinghua）的庄重感 + 现代 SaaS 产品的交互精致度。用深靛蓝（School Blue）为骨架，暖金为点睛，大面积留白与精致文字排版营造高级感。

## 五、动效系统

### 5.1 动效原则
- **克制优雅** — 动效服务于内容，不喧宾夺主
- **物理真实** — 缓动曲线模拟真实物体运动
- **有目的性** — 每个动效都有信息传递功能

### 5.2 缓动曲线
| 名称 | 曲线 | 用途 |
|------|------|------|
| `ease-out-expo` | `cubic-bezier(0.16, 1, 0.3, 1)` | 元素入场、页面切换 |
| `ease-in-out-smooth` | `cubic-bezier(0.65, 0, 0.35, 1)` | 悬停、状态变化 |
| `ease-spring` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹性反馈、按钮点击 |

### 5.3 核心动效组件

#### A. 校训文字流（MottoStream）
- **位置**：首页Hero区底部、页脚上方
- **效果**：校训"本固枝盛，学富国强"以书法笔触方式逐字显现，金色光晕流动
- **技术**：CSS `@keyframes` + `background-clip: text` + 金色渐变动画
- **时长**：单字显现 0.8s，间隔 0.15s，整体循环 12s

#### B. 学院粒子场（AcademicParticles）
- **位置**：Hero背景、登录页左侧
- **效果**：靛蓝色粒子缓慢浮动，偶尔连接成细线（模拟知识网络）
- **技术**：Canvas 2D，`requestAnimationFrame`
- **参数**：粒子数 40，速度 0.3px/f，连线距离 120px，连线透明度 0.08

#### C. 卷轴揭示（ScrollReveal）
- **位置**：全站各区块
- **效果**：元素从下方 30px 淡入上浮，带轻微模糊消除
- **触发**：IntersectionObserver，threshold 0.15
- **时长**：0.6s，`ease-out-expo`
- **错开**：同组元素间隔 0.1s

#### D. 磁性悬停（MagneticHover）
- **位置**：导航链接、重要按钮、版块卡片
- **效果**：鼠标靠近时元素轻微向光标方向偏移（最大 8px）
- **技术**：`onMouseMove` 计算偏移，`transform: translate()`
- **回弹**：鼠标离开时 0.4s 弹性回归

#### E. 数字滚动（CountUp）
- **位置**：首页统计区
- **效果**：数字从 0 滚动到目标值，带缓动
- **时长**：2s，`ease-out-expo`
- **格式化**：千位分隔，到达后轻微放大 1.05x 后回弹

#### F. 卡片光泽扫过（CardShine）
- **位置**：帖子卡片、版块入口
- **效果**：鼠标悬停时，一道白色半透明光带从左上到右下扫过
- **技术**：伪元素 `::after` + `transform: translateX()` + `skewX(-15deg)`
- **时长**：0.6s

### 5.4 页面转场
- **类型**：共享元素转场（View Transition API）
- **效果**：页面切换时，共同元素（如标题、图片）平滑过渡
- **降级**：不支持浏览器使用淡入淡出 0.3s

## 十一、暗色模式策略

- **切换方式**：手动切换（保留现有）+ 系统偏好检测
- **核心映射**：
  - 白色背景 → `indigo-950`
  - 浅灰背景 → `indigo-900`
  - 深色文字 → `slate-100`
  - 边框 → `indigo-800`
- **特殊处理**：
  - 金色在暗色下更亮（`gold-400`）
  - 分类色保持但降低饱和度
  - 粒子效果白色更亮
